# Security Policy

## Reporting a Vulnerability
Please report security issues privately:
- Email: security@bab3l.dev (or open a private advisory if enabled)
- Provide NetBox version, SDK version, and reproduction details.

## Scope
This SDK is a client library. Vulnerabilities typically relate to:
- Dependency CVEs
- Request/response handling (e.g., header injection)
- TLS configuration misuse

## Handling
We will triage within 5 business days and publish patches via tagged releases. Dependabot maintains dependency updates.
